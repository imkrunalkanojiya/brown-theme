A lightweight, simple blog theme for bloggers.

== Description ==

A basic, plain-vanilla, lightweight theme, best suited for building your site blogging purpose only.

== Installation ==

1. In your site's admin panel, go to Appearance > Themes and click `Add New`.
2. Type "Brown Theme" in the search field.
3. Click `Install` and then `Activate` to start using the theme.
4. Enjoy :)
